Add a metadata setting to this set of remote session settings.

Parameters

settingName string

The name of the setting to set.

settingValue object

The value of the setting.

Remarks

The value to be set must be serializable to JSON for transmission across the wire to the remote end. To be JSON-serializable, the value must be a string, a numeric value, a boolean value, an object that implmeents IEnumerable that contains JSON-serializable objects, or a Dictionary<TKey, TValue> where the keys are strings and the values are JSON-serializable.

Exceptions

ArgumentException

Thrown if the setting name is null, the empty string, or one of the reserved names of metadata settings; or if the setting value is not JSON serializable.
