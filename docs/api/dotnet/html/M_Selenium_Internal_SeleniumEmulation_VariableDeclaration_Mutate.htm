VariableDeclarationMutate Method WebDriver
Mutate a script so that variable declarations have the correct scope. The original, unmodified script is used to generate a script on the StringBuilder, the "ToString" method of which should be used to get the result. We make use of a StringBuilder rather than a normal String so that we can efficiently chain mutators.
