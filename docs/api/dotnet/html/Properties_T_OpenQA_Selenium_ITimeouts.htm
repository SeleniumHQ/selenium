ITimeouts PropertiesWebDriver

The ITimeouts type exposes the following members.

Properties
  NameDescription
AsynchronousJavaScript
Gets or sets the asynchronous script timeout, which is the amount of time the driver should wait when executing JavaScript asynchronously. This timeout only affects the ExecuteAsyncScript(String, Object) method.
ImplicitWait
Gets or sets the implicit wait timeout, which is the amount of time the driver should wait when searching for an element if it is not immediately present.
PageLoad
Gets or sets the page load timeout, which is the amount of time the driver should wait for a page to load when setting the Url property.
