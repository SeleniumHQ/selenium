org.openqa.selenium

Annotation Type Beta


  • @Retention(value=RUNTIME)
@Target(value={ANNOTATION_TYPE,CONSTRUCTOR,FIELD,METHOD,TYPE})
@Documented
public @interface Beta
    Indicates that a feature or API is in active development, and so should not be relied upon. The update policy for anything marked beta is that it may be deleted in the next Selenium release without warning. In the ideal world, this would cause the method to blink gently in the user's IDE. We don't live in the ideal world. We'll find out the hard way whether reading docs is the same thing.
