com.thoughtworks.selenium.condition

Class Condition

  • java.lang.Object
    • com.thoughtworks.selenium.condition.Condition
  • Direct Known Subclasses:
    Not, Presence, Text

    public abstract class Condition
extends java.lang.Object
    Simple predicate class, which also knows how to wait for the condition to be true. Used by Selenium tests.

    Conditions have two basic properties:

    • a message (purely used for displaying purposes)
    • an implementation of isTrue(com.google.testing.selenium.condition.ConditionRunner.Context)
