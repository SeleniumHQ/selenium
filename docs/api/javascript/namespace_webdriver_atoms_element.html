code »webdriver.atoms.element.getAttribute (element, attribute) ⇒ ?string

Get the value of the given property or attribute. If the "attribute" is for a boolean property, we return null in the case where the value is false. If the attribute name is "style" an attempt to convert that style into a string is done.

Parameters

The element to use.
