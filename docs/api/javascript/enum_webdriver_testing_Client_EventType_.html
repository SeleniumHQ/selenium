Enum webdriver.testing.Client.EventType_

code »
Type: string

The types of events that may be published by a TestClient to the server.

Values and Descriptions

ERROR
Sent when an error has occurred. The data object will have the following fields: - message: The error message.
INIT
Sent to signal that a test suite has been fully initialized.
RESULTS
Sent when all tests have completed. The data object will have the following fields: - isSuccess: Whether the tests succeeded. - report: A flat log for the test suite.
SCREENSHOT
Sent when there is a screenshot for the server to record. The included data object will have two fields: - name: A debug label for the screenshot. - data: The PNG screenshot as a base64 string.
START_TEST
Sent when starting a new test. The data object will have the following fields: - name: The name of the test.
