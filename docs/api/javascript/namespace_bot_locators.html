code »bot.locators.findElement (target, opt_root) ⇒ Element

Find the first element in the DOM matching the target. The target object should have a single key, the name of which determines the locator strategy and the value of which gives the value to be searched for. For example {id: 'foo'} indicates that the first element on the DOM with the ID 'foo' should be returned.

Parameters
target: !Object
The selector to search for.
opt_root: (Document|Element)=
The node from which to start the search. If not specified, will use document as the root.
Returns
The first matching element found in the DOM, or null if no such element could be found.
code »bot.locators.findElements (target, opt_root) ⇒ !goog.array.ArrayLike.<Element>

Find all elements in the DOM matching the target. The target object should have a single key, the name of which determines the locator strategy and the value of which gives the value to be searched for. For example {name: 'foo'} indicates that all elements with the 'name' attribute equal to 'foo' should be returned.

Parameters
target: !Object
The selector to search for.
opt_root: (Document|Element)=
The node from which to start the search. If not specified, will use document as the root.
Returns
All matching elements found in the DOM.

code »bot.locators.STRATEGIES_ : Object.<string, bot.locators.strategy>

Known element location strategies. The returned objects have two methods on them, "single" and "many", for locating a single element or multiple elements, respectively. Note that the versions with spaces are synonyms for those without spaces, and are specified at: https://code.google.com/p/selenium/wiki/JsonWireProtocol
