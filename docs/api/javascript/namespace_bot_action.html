code »bot.action.multiTouchAction_ (element, transformStart, transformHalf, opt_coords, opt_touchscreen)

Performs a multi-touch action with two fingers on the given element. This helper function works by manipulating an "offsetVector", which is the vector away from the center of the interaction at which the fingers are positioned. It computes the maximum offset vector and passes it to transformStart to find the starting position of the fingers; it then passes it to transformHalf twice to find the midpoint and final position of the fingers.

Parameters
element: !Element
Element to interact with.
transformStart: function(goog.math.Vec2)
Function to transform the maximum offset vector to the starting offset vector.
transformHalf: function(goog.math.Vec2)
Function to transform the offset vector halfway to its destination.
opt_coords: goog.math.Coordinate=
Position relative to the element at the center of the pinch.
opt_touchscreen: bot.Touchscreen=
Touchscreen to use; if not provided, constructs one.

code »bot.action.scrollIntoView (element, opt_region) ⇒ boolean

Scrolls the given element in to the current viewport. Aims to do the minimum scrolling necessary, but prefers too much scrolling to too little. If an optional coordinate or rectangle region is provided, scrolls that region relative to the element into view. A coordinate is treated as a 1x1 region whose top-left corner is positioned at that coordinate.

Parameters
element: !Element
The element to scroll in to view.
opt_region: !(goog.math.Coordinate|goog.math.Rect)=
Region relative to the top-left corner of the element.

code »bot.action.type (element, values, opt_keyboard, opt_persistModifiers)

Types keys on the given element with a virtual keyboard.

Callers can pass in a string, a key in bot.Keyboard.Key, or an array of strings or keys. If a modifier key is provided, it is pressed but not released, until it is either is listed again or the function ends.

Example: bot.keys.type(element, ['ab', bot.Keyboard.Key.LEFT, bot.Keyboard.Key.SHIFT, 'cd']);

Parameters
element: !Element
The element receiving the event.
values: (string|!bot.Keyboard.Key|!Array)
Value or values to type on the element.
opt_keyboard: bot.Keyboard=
Keyboard to use; if not provided, constructs one.
opt_persistModifiers: boolean=
Whether modifier keys should remain pressed when this function ends.
Throws
bot.Error
If the element cannot be interacted with.
