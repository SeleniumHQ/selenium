Namespace bot.geolocation

Global Functions

code »bot.geolocation.getCurrentPosition (successCallback, opt_errorCallback, opt_options)

Provides a mechanism to retrieve the geolocation of the device. It invokes the navigator.geolocation.getCurrentPosition method of the HTML5 API which later callbacks with either position value or any error. The position/ error is updated with the callback functions.

Parameters
successCallback: function(?GeolocationPosition)
The callback method which is invoked on success.
opt_errorCallback: function(GeolocationPositionError)=
The callback method which is invoked on error.
opt_options: GeolocationPositionOptions=
The optional parameters to navigator.geolocation.getCurrentPosition; defaults to bot.geolocation.DEFAULT_OPTIONS.

Global Properties

code »bot.geolocation.DEFAULT_OPTIONS : GeolocationPositionOptions

Default parameters used to configure the geolocation.getCurrentPosition method. These parameters mean retrieval of any cached position with high accuracy within a timeout interval of 5s.
